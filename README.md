# Kashmiri Food Website

A full-stack restaurant web application where users can explore food items, place orders, and manage their accounts. The system includes an admin panel for managing menu items, orders, and overall platform functionality.

---

##  Features

###  User Side
- User registration and login system  
- Browse food items and categories  
- Add items to cart  
- Place orders بسهولة (Cash on Delivery / Wallet system)  
- Wallet system for payments  
- View and manage orders  

###  Admin Panel
- Add new food items  
- Update existing items  
- Delete food items  
- Manage orders  
- Control website content dynamically  

---

##  Core Functionalities

- Full cart management system  
- Order placement and tracking  
- Payment options:
  - Cash on Delivery  
  - Wallet system  
- Authentication system (Login/Signup)  
- Dynamic data handling with backend
